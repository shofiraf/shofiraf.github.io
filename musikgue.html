MUSIKGUE

Project

WHAT IS MUSIKGUE

MusikGue is a mobile-based music player application with Apple and Android Operating Systems. MusikGue has collaborated with many labels and provide music with various category.

MUSIKGUE'S FEATURES

MusikGue has a modern, flexible, and easy to understand design. MusikGue helps users to find the right songs quickly with search and playlist features.

WHY I CREATED THIS PROJECT

MusikGue is an individual project and created to complete the midterm exam of Advanced Information System Analysis and Design Subject.

MY CONTRIBUTION TO THE PROJECT

As an individual project, my contribution to MusikGue Project is to design User Interface for all features in MusikGue application.
