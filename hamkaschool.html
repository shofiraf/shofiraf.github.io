WHAT IS HAMKA SCHOOL

Hamka School is a high school located in Central Java. Hamka School website was created to promote Hamka Schools to the public and contains information about events, academic calendars, articles, etc.

HAMKA SCHOOL'S FEATURE

Hamka School provides all information about the school and student activities such as articles, events, academic calendars, alumni testimonials, etc.

WHY I CREATED THIS PROJECT

Hamka School is a group case study project consisting of two members and created to complete the assignment of User Experience Subject.

MY CONTRIBUTION TO THE PROJECT

My contribution to this project is to conduct a brainstorming, design User Interface, and create User Experience for all features based on the application's flow using Axure RP8.
